# Implementation Plan: Access Token Expiry Alignment

**Branch**: `017-token-expiry-alignment` | **Date**: 2026-03-15 | **Spec**: [spec.md](./spec.md)
**Input**: Update `expires_in` to 1800s and make it configurable via `sharedConfig.SECURITY`.

## Summary
Align the access token lifespan with Singpass specifications (1800 seconds / 30 minutes) by introducing a new configuration parameter in the shared package and updating the backend token service to use it.
