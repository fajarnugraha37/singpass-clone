# vibe-auth

This project is a monorepo for the vibe-auth application.

## Frontend

The frontend is an Astro application located in `apps/frontend`.

## Backend

The backend is a Hono application located in `apps/backend`.

### Serving Frontend Assets

The backend is responsible for serving the static assets generated by the frontend. This is handled by a middleware that serves files from the `apps/frontend/dist` directory. The middleware also handles client-side routing by serving `index.html` as a fallback for any requests that don't match a static file.
