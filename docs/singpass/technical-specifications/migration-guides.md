# Migration Guides

This section contains guides to help you migrate your existing integrations to be compliant with the new Authentication API.

If you are unsure about the type of your app, you may log in to the [Singpass Developer Portalarrow-up-right](https://developer.singpass.gov.sg/). The app types of each of your apps will be visible on the dashboard page, once you log in.
